## 배우는 알고리즘 문제 해결 전략 - 1

### 06 무식하게 풀기 : Brute-force 

: 컴퓨터의 빠른 계산 능력을 이용해 가능한 경우의 수를 일일이 나열하며 문제 해결

- 재귀 함수 : recursion

  - 자신이 수행할 작업을 유사한 형태의 여러 조각으로 쪼갠 뒤

    자신이 그 중 한 조각을 수행하고, 

    나머지를 자기 자신을 호출해 실행하는 함수

  - 최소한의 작업에 도달했을 때

    답을 곧장 반환하는 재귀 호출의 기저 사례( base case ) 필수적

    => 반복적인 코드 제거에 유용

- 완전 탐색 : Exhaustive search
